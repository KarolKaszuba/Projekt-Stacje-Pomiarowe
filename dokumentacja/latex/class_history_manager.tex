\doxysubsection{Detailed Description}
Klasa zarządzająca historią sesji aplikacji. 

Klasa \doxylink{class_history_manager}{History\+Manager} odpowiada za zapisywanie, wczytywanie i aktualizowanie danych sesji, takich jak informacje o stacjach pomiarowych, sensorach, pomiarach i jakości powietrza. Dane są przechowywane w plikach JSON w określonym katalogu.
